## Download

Grab via Gradle:

```groovy
compile 'com.dzaitsev.rxviper:rxviper:1.0.0-rc3'
```

Maven:

```xml
<dependency>
    <groupId>com.dzaitsev.rxviper</groupId>
    <artifactId>rxviper</artifactId>
    <version>1.0.0-rc3</version>
</dependency>
```
